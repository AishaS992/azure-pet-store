Please note, The Azure Pet Store is a hypothetical open source application developed to help you learn about and build awesome software on Azure. This is not official Microsoft documentation but rather an opinionated approach/guidance to help you get started. It is comprised of the Pet Store App and several Pet Store Microservices that are referenced in the Azure Pet Store Reference Architecture located here. Covered are the many services in Azure such as CI/CD, TDD, Automated Testing, Container Strategy with App Services and AKS, Correlated Telemetry, Power Apps, Serverless Functions, Logic Apps and much more.
